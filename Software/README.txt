To run the whole software part, follow steps below:

1. Run 'Aggregate_features' in 'BIDMIC_Aggregation.py' under 'BIDMC_Feature_Aggregation' directory.
   
   'BIDMC_Segmented_features.csv' will be generated as its output.

2. With this file, run ReliefF and GPR according to testing code in 'BIDMC_Regression' directory.
